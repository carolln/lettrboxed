## PROJETO FINAL: LP2 - JAVA
_Alunos: Carolina Nunes & Mateus Oliveira_

### Proposta
A API em questão tem como objetivo concentrar, gerenciar e organizar avaliações de filmes de forma prática para o usuário - buscando, acima de tudo, resolver a descentralização de materiais na internet que tangem à bolha cinematográfica e a ausência de aplicações direcionadas. Assim, essa API busca representar um centro de informações, direcionada tanto para usuários especificamente interessados na esfera do cinema quanto para usuários casuais que buscam recomendações com base na opinião pública.
